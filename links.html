Physics Software

This is very brief list of computational physics packages that I use routinely in my research:

  • FHI-aims: All-electron full-potential electronic structure code with numeric atom-centered basis functions developed by the Fritz-Haber-Institute.
  • Quantum Espresso: Suite for electronic structure calculations based on plane-waves and pseudopotentials.
  • BerkeleyGW: Parallel GW and Bethe-Salpeter equation code developed by UC Berkeley.
  • Kwant: Tight-binding Python package for quantum transport simulations.
